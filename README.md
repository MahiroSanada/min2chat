min2chatに便利機能を追加したものです。機能としては

- 削除スピード制限なし
- 文字数制限なし
### 時刻の設定
時刻の設定というものがあり、時刻の形式を変更することができます。
- 普通(2026/01/10 19:33)
- 改造 (19:33:46)
- システム (2026-01-10T10:32:17.538Z)
- 任意 (自由に記述可)

### リアクション設定
リアクション設定です。
このボタンを押して、一番上の欄に送りたいリアクションを入力します。複数を送りたいときは、カンマ区切りです。例：こんにちは,👋,😠
そして投稿に表示される、⚙のアイコンをクリックし、送りたいリアクションをクリックすると送れます。(注意　送られた側はページを更新しないと見えないことがあります。)
そして、リアクション設定ボタンをクリックして、一番下の欄には、一回に増えるリアクションの数を指定できます。
